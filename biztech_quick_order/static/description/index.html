Product Orders In Bulk Managed Quickly

Are you looking for an app that allows your Odoo e-commerce store customers to order products in bulk? AppJetty has a solution in the form of Odoo Matrix Quick Order App that lets your customers order products in bulk. With Matrix Quick Order admin can allow e-commerce store customers to order items in bulk by providing them option to add and manage numerous quick order lists. Customers will maintain the list and order items in bulk as per the requirement. With the pre generated list, admin will be able to save time for customers, who order multiple items in bulk on regular occasions. Customers can add as many products as required in the order list for a rapid ordering process.

Core Features

Admin has full control over Quick Order app. Admin can also manage following things from the back end.

  • Title : Title for the quick order option.

  • Short Message : Short Message to display with Quick order.

  • Maximum Products allowed on one page : To set maximum products displayed on single page of List.

  • Do you want to enable multiple list? : Enable it to let customer manage multiple Quick order lists.

Customer can Create and manage multiple lists

  • To create a new list go to User Profile Area on shop page and click on Quick Order to manage Quick Order List.

  • To create a new list click on Create New List button.

  • After creating a new list click on Edit button to add products to the list. Click on Move to list to move specific item into the Quick order list.

  • Customer can also add product to the list from Product page itself. Go to any product and click on Add to List button. And choose from all available lists.

Customer can place order directly from the quick order list

  • After adding products to the list go to User Profile Area -> Quick Order and click on the created list.

  • Add required product quantity and click on Add to Cart to place order with Matrix Quick Order.
